Sources
----------

`Related Chrome Dev Docs Page <https://developers.google.com/web/updates/2015/05/go-to-a-line-number-at-a-specific-column>`_

For a large file like a bundeled webpack file, go to ``sources`` and click on the source file. The press ``CTRL`` + ``O`` and type in the line number and column in the format ``<line number>:<column number>`` (`e.g.` ``8224:57``).
