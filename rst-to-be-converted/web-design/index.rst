FluidVids
~~~~~~~~~~~

`FluidVids <fluidVids_>`_ is a js library that makes most common video formats responsive. Make sure to use the code in the GitHub because it's the most up to date.

.. _fluidVids: https://toddmotto.com/fluid-and-responsive-youtube-and-vimeo-videos-with-fluidvids-js/
