User Preferences
------------------

To avoid accidently recoppying when you are trying to paste, use this setting in user preferences. It will prevent ``Ctrl`` + ``C`` from coppying when there is an empty selection.

.. code-block:: json

	{
		"copy_with_empty_selection": "false"
	}
